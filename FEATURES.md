### M6 带宽利用率

- 前端：接口表格利用率列已有颜色预警，需补齐 ifSpeed 字段并在流量图中展示带宽上限
- 后端：`database.get_latest_interfaces()` 和 `get_interface_traffic()` 可按采集间隔 delta 算真实 bps 利用率

### M2 告警通知（站内信）

- 顶栏告警铃铛已有 badge 计数，需改为轮询或 WebSocket 实时更新
- 告警触发时在页面内弹出通知（非浏览器原生 alert）
- 告警列表页已可用，补齐实时铃铛

### A4 设备标签

- devices 表已有 tags 字段，前端加标签输入/展示
- 设备列表支持多标签筛选

### A6 IP 地址管理 (IPAM)

- 新建 `ip_subnets` 和 `ip_allocations` 表
- IP 段管理：添加网段、查看利用率
- IP 分配/回收：绑定到设备接口
- 新页面 `/ipam`

### A8 配置备份

- 定时从设备拉取 running-config 存库
- 版本对比（diff 视图）
- 变更记录

### C1/C2/C4 端口控制

- 接口操作页面：选择设备 → 接口列表 → shutdown/undo/description/VLAN
- 后端通过 SSH `execute_commands()` 下发
- 复用 driver 抽象层

### C8 配置模板

- Jinja2 模板管理：预定义模板 + 自定义变量
- 渲染后预览 + 下发
