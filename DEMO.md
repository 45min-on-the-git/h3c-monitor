# H3C Monitor 演示路线图

## 第一幕：全局态势感知（领导视角）

### 1. NOC 大屏 `/bigscreen`

全屏深色页面，10 秒自动刷新。总设备数、在线数、活跃告警数三大数字 + CPU/内存仪表条 + 告警列表。一屏看清全网状态。

### 2. 监控总览 `/`

设备卡片（CPU/内存实时值，超标自动变红）→ 指标趋势图（选单设备看 CPU/内存/温度历史曲线）→ 接口状态表（搜索、筛选、点接口行弹出流量时序图）。从全局大屏下钻到单设备细节。

### 3. 告警中心 `/alarms`

CPU>80%、内存>90%、温度>60 自动触发告警，指标恢复自动消除。顶栏铃铛实时轮询（30s），新告警弹 toast 提醒。可确认/查看历史。

### 4. 定时报表 `/reports`

选设备+时间范围（1/7/30天），CPU/内存/温度的平均值/最大值聚合，适合周会报数据。

## 第二幕：资产与资源管理（运维视角）

### 5. 设备台账 `/devices`

每台设备记录序列号、采购日期、维保到期、供应商、机房、机柜、标签。支持按机房/标签/类型筛选，搜索。点击编辑弹窗修改全部字段。

### 6. IP 地址管理 `/ipam`

左侧子网列表（名称/网段/CIDR/网关/VLAN）→ 点击展开右侧 IP 分配网格。标记 used/free/reserved，记录绑定设备和接口。

### 7. VLAN 台账 `/vlan-registry`

选设备 → 点「同步设备 VLAN」→ 自动从设备读取 VLAN 列表写入台账。支持补充用途说明。

---

## 第三幕：变更控制与审计（安全/合规视角）

### 8. 变更审计 `/audit`

所有操作自动记录：谁在什么时间对哪个设备做了什么。支持端口启停、ACL变更、VLAN创建/删除、配置备份、路由修改等全程溯源。

### 9. 配置备份 `/config`

选设备 → 立即备份（通过 SSH/NETCONF 拉 running-config，存库带 hash）→ 备份历史列表 → 点击查看内容 → 支持回滚（把某次备份写回设备）。

---

## 第四幕：设备配置控制（高级运维视角）

### 10. 端口管理 `/interfaces`

选设备 → 接口列表（含速率/带宽/利用率列）。每行三个操作按钮：shutdown/undo、修改描述、VLAN配置（access绑定VLAN ID 或 trunk 配 PVID+允许列表）。

### 11. ACL 管理 `/acl`

选设备 → 自动 SSH 读取 `display acl all` 解析展示 → 新建规则（ACL编号/规则ID/动作/协议/源/目的）→ 下发到设备 → 可编辑/删除。

### 12. 静态路由 `/routes`

选设备 → 读取静态路由表 → 添加（目的网段+掩码+下一跳+优先级）→ 删除。支持日常路由运维。

### 13. 配置模板 `/templates`

写 Jinja2 模板（如标准端口配置）→ 实时预览渲染结果 → 保存模板库。供批量下发复用。

## 技术演进对比

| 维度 | 初始版 | 当前版 |
|------|--------|--------|
| 采集方式 | SSH CLI 正则 | SNMP 优先 → SSH 兜底 |
| 配置下发 | 无 | NETCONF 优先 → SSH 兜底 |
| 页面 | 1 个单页 | 15 个页面（侧边栏导航） |
| 告警 | 无 | 规则引擎 + 自动触发/恢复 + 铃铛推送 |
| 配置控制 | 无 | 端口/VLAN/ACL/路由 4 类 12 种操作 |
| 台账 | 无 | 设备/IP/VLAN/端口分配 4 类台账 |
| 安全合规 | 无 | 变更审计 + 配置备份 + 回滚 |
| 测试 | 0 | 18 个单测（ACL解析/告警引擎/数据库） |
| 主题 | 大白底 | 明/暗双主题 + NOC 暗色大屏 |
